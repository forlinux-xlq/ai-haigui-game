# AI海龟汤游戏 PRD

## 1. 产品概述
AI驱动的海龟汤推理游戏网站。玩家与AI主持人对话，通过“是/否/无关”回答机制，逐步推理出离奇故事背后的真相。

## 2. 目标用户
- 推理游戏爱好者
- 寻求轻松娱乐的上班族
- AI应用尝鲜者

## 3. 核心功能

### 3.1 MVP功能

#### 游戏大厅
- 海龟汤列表（标题、难度标签、类型标签、简短导语）
- 筛选功能（按难度：简单/中等/困难；按类型：悬疑/恐怖/温情/科幻）
- 开始游戏入口

#### 游戏页面
- 汤面故事固定展示区
- 聊天输入框（玩家自然语言提问）
- AI回答区（严格限定：“是”/“否”/“无关”）
- 对话历史时间轴
- “查看汤底”按钮（主动揭晓真相）
- “结束游戏”按钮

#### 汤底页面
- 完整汤底故事展示
- 玩家推理过程回顾（提问轮次统计）
- “再来一局”按钮（返回大厅）

### 3.2 后续迭代
- 用户登录与账号系统
- 游戏记录存档与历史回顾
- 推理排行榜（最少提问轮次榜）
- UGC自定义海龟汤（玩家创作审核入库）
- 多人对战模式（竞速推理）

## 4. 界面设计

### 整体风格
神秘悬疑氛围，深蓝色调（#0f172a为主色），金色点缀（#f59e0b）

### 游戏大厅
卡片式网格布局，悬停效果，显示故事标题、难度标签、类型图标

### 游戏页面
类即时通讯界面设计：
- 顶部固定汤面区域
- 中部滚动对话区（玩家右气泡/AI左气泡）
- 底部固定输入框

### 汤底页面
全屏揭晓动效，真相文字逐行显现，高亮关键反转信息

## 5. 技术栈
- 前端：React + TypeScript + Tailwind CSS
- 后端：Node.js + Express
- AI服务：DeepSeek / Claude API（负责语义理解与逻辑判定）
- 部署：Vercel
