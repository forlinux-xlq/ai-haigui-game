# AI海龟汤游戏需求文档（Research）

## 1. 项目概述
基于AI的海龟汤在线推理游戏，玩家通过与AI主持人对话，推理出离奇故事背后的真相。

## 2. 核心概念
- **汤面**：故事的表面描述，看似离奇或矛盾
- **汤底**：故事的完整真相
- **主持人**：AI扮演的游戏引导者，仅回答“是”、“否”、“无关”

## 3. 功能需求

### 3.1 汤面展示
- AI随机或按难度提供汤面故事
- 支持多种故事类型（恐怖/悬疑/科幻/温情等）

### 3.2 问答系统
- 玩家输入问题
- AI根据汤底判断并回答“是”/“否”/“无关”

### 3.3 对话记录
- 保存玩家提问与AI回答历史
- 支持查看完整对话流程

### 3.4 汤底查看
- 玩家可随时选择查看汤底真相
- 支持猜中后自动展示完整答案

## 4. 游戏规则
1. 主持人仅回答“是”、“否”、“无关”
2. 玩家通过提问逐步推理真相
3. 猜中核心真相或主动请求可查看汤底

## 5. 技术要点
- **AI语义理解**：准确解析玩家问题意图
- **逻辑判定**：基于汤底严格判断答案
- **上下文记忆**：保持多轮对话连贯性
